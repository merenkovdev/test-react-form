# test-react-form

Форма редактирования постов.

## Запуск

Установить все необходимые зависимости

```shell
npm install
```

Запуска проекта в режиме разработки,
проект автоматически откроется в новой вкладке браузера

```shell
npm run dev
```

Версия для продакшн

```shell
npm run build
```

В папке *dst* сохранен собранный проект.
